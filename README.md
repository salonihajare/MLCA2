# 📱 Mobile Price Predictor

A machine learning-powered web application that predicts the price of a smartphone based on its specifications like RAM, ROM, mobile size, camera details, and battery.

## 🚀 Features

- ML model trained with Random Forest Regressor and tuned via GridSearchCV.
- Predicts smartphone price based on user input.
- Simple HTML frontend + Flask backend.
- Model performance evaluated using MAE, RMSE, and R².

## 🔧 Technologies Used

- Python
- Pandas
- Scikit-learn
- Flask
- HTML, CSS, JavaScript
